# FPGA Projectile Game

## Overview
A two-player projectile game implemented in Verilog on a Spartan-7 FPGA.

## Features
- Collision detection
- Score tracking
- Debounced button inputs
- LED projectile visualization

## Hardware
- Spartan-7 FPGA
- LEDs
- Push buttons

## Skills Demonstrated
- Verilog
- Finite State Machines
- Digital Logic Design
- Hardware Debugging
